# 🎬 프로젝트 소개
우리는 하루에도 수 많은 일들을 합니다.
할 일을 쉽게 기록하고 얼마나 잘 했는지 확인할 수 있다면 얼마나 좋을까요?
Planit은 할 일을 완료할 때마다 행성이 자라나면서 동기부여를 제공하는 To-do list 서비스입니다.

계획적인 하루를 보내고 싶으신가요?
To-do list를 작성만 하고 실행하지 않은 적이 있나요?
Planit에서 To-do를 완료하고 행성을 키워보세요!
